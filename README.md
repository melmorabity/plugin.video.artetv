Arte video plugin for Kodi
==========================

This add-on provides access to the [Arte](https://www.arte.tv/) live and catchup TV platform.

Dependencies
------------

* Kodi 18 or higher
* script.module.dateutil
* script.module.inputstreamhelper
* script.module.requests
